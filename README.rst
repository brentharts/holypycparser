===============
holypycparser
===============
based on pycparser v2.22


holysh.py
------------------
example terminal shell

```
python3 holysh.py [MY.HC]
```


holypycparser.py
------------------
convert C to HolyC and HolyC to C

```
python3 holypycparser.py [MY.C] --out=[MY.HC]
python3 holypycparser.py [MY.HC] --out=[MY.C]

```
